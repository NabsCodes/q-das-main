# Q-DAS Global Group

A modern Next.js website for Q-DAS Global Group, showcasing our mission to build Africa's digital, energy, and infrastructure future. The platform highlights our three core focus areas: Technology (digital transformation and cybersecurity), Energy (renewables and industrial power systems), and Infrastructure (construction and real estate development).

## Overview

Q-DAS Global Group is committed to transforming Africa's technological landscape through innovative solutions in digital infrastructure, sustainable energy, and construction development. This website serves as our digital presence, communicating our vision and capabilities to partners, clients, and stakeholders.

## Technology Stack

- **Framework**: Next.js 16.0.3
- **React**: 19.2.0
- **Styling**: Tailwind CSS 4
- **UI Components**: Radix UI, Shadcn UI
- **Animations**: Motion
- **Form Handling**: React Hook Form with Zod validation
- **Type Safety**: TypeScript 5

## Development Setup

### Prerequisites

- Node.js 20 or higher
- pnpm (recommended) or npm/yarn

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd q-das-main
```

1. Install dependencies:

```bash
pnpm install
```

1. Run the development server:

```bash
pnpm dev
```

1. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

### Available Scripts

- `pnpm dev` - Start the development server
- `pnpm build` - Build the application for production
- `pnpm start` - Start the production server
- `pnpm lint` - Run ESLint to check code quality

## Deployment

This application is optimized for deployment on modern platforms such as Vercel, Netlify, or any Node.js hosting service. The production build can be generated using:

```bash
pnpm build
```

For detailed deployment instructions, refer to the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying).
